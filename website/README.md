# Reviewate Website

Marketing and landing page for Reviewate, built with Nuxt 4.

## Tech Stack

- [Nuxt 4](https://nuxt.com/) — Vue meta-framework
- [Nuxt UI](https://ui.nuxt.com/) — Component library
- [Nuxt Content](https://content.nuxt.com/) — Markdown-driven content
- [Motion Vue](https://motion.vueuse.org/) — Animations

## Development

```bash
# From the monorepo root
cd website

# Install dependencies (if not already done from root)
pnpm install

# Start dev server
pnpm dev
```

The site runs at `http://localhost:3000` by default.

## Build

```bash
pnpm build     # Build for production
pnpm generate  # Generate static site
pnpm preview   # Preview production build locally
```
